Android高级笔记

一、在子线程能否更新UI？

在 ViewRoot 还没有被创建的时候，是可以在子线程中直接更新 UI。

  • ViewRoot 在 onCreat 时没有创建
  • ViewRoot 在 onResume 后才创建
  • UI 线程是没有加锁的，在非 UI 线程更新 UI 时候是不安全的操作，所以 Android 不让在子线程中直接更新 UI。
